Client Catalogo MetaSearch¶

MetaSearch è un plugin di QGIS per interfacciarsi con i servizi di catalogazione metadati, con il supporto dello standard Catalogue Service for the Web (CSW) dell’OGC.

MetaSearch offre un approccio semplice ed intuitivo con un’interfaccia user-friendly per effettuare ricerche in cataloghi di metadati all’interno di QGIS.
